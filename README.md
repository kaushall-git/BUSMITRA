# Bus Mitra 🚌

![Bus Mitra Logo](assets/images/bus_mitra_logo.png) <!-- Add your logo path -->

**Bus Mitra** is a comprehensive public transport management system designed to enhance the commuting experience for both passengers and staff. The app provides real-time bus tracking, ticket booking, route information, and vehicle condition monitoring.

## Features ✨

### For Passengers 👥
- 📍 Real-time bus location tracking
- 🎫 Digital ticket booking & management
- 🗺️ Interactive route maps and schedules
- ⚠️ Emergency reporting system
- 💰 Digital wallet integration
- 📱 Profile management

### For Staff (Drivers/Conductors) 🚌
- 📍 Live location sharing
- 👥 Passenger count management
- 🎟️ Ticket verification system
- 📊 Sustainability reports
- ⏰ Schedule management
- 📝 Incident reporting

## Technologies Used 💻

- **Frontend**: Flutter (Dart)
- **Backend**: Firebase (Realtime Database, Authentication)
- **Maps**: Google Maps API
- **State Management**: Provider
- **Local Storage**: SharedPreferences
- **Other Packages**:
  - `geolocator` - Location services
  - `google_maps_flutter` - Map integration
  - `image_picker` - Profile pictures
  - `intl` - Date/time formatting
